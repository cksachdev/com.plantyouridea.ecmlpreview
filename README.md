# com.plantyouridea.ecmlpreview
Preview ecml on top of the editor

Refer to google document link below for complete requirement specification:
https://docs.google.com/document/d/14CizhzrHFdZAqGlpPKMY2ssrD2zhr8bMDmqsOym3PxU/edit

Problem:
Editor used fabricJS and the preview is using createJS library. It is difficult to guess, if what is rendered on editor is rendering in same dimensions as Editor. This plugin addresses this problem and allows developer to check it by putting an overlay on top of editor canvas.
